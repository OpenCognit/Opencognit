[Showcase] OpenCognit — Open-source AI agent orchestration OS

Hey r/selfhosted,

We've been building OpenCognit for a while and just hit a milestone where we need real-world testing. It's not a chatbot wrapper — it's a full OS for running autonomous agent teams.

What it does:
• CEO agent with Extended Thinking breaks down goals
• Creates specialist agents (dev, research, writer, etc.)

• Task DAG with auto-unlocking dependencies
• Multi-user with role-based access
